## About SG Double

Description: SG Double is a multipurpose theme for the blog, portfolio or e-commerce websites. It has a lot of options in the customizer, 4 page templates, custom header, custom background, layout choices (right/left/two columns or full width) for all pages according to a hierarchy, custom width for columns, per page sidebars, social icons, logo, Google fonts, CSS3+JQuery animation and more. Translation ready and supports portfolio custom post type from Jetpack and WooCommerce. See the demo at http://wpblogs.info/demo/sgdouble/

## How to install

* Unzip and then copy the sgdouble directory to /wp-content/themes/ in your WordPress installation. 
* Activate from admin menu. 

## How to use

* Theme options placed at "Appearance >> Customize"
* Widgets can be found at "Appearance >> Widgets"
